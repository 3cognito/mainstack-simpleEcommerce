# Mainstack-simple-ecommerce-api

This is a simple CRUD API written using Nodejs, Typescript, Express and MongoDB, with the following features:

1. Create, get all and get single product
2. Update product
3. Delete product
4. Simple pagination

## Run application in development

To run the application in a development environment
- Clone this repo
- Create a ".env" file in the root directory, the file will should contain two values DB_URI="a valid mongodb connection string", NODE_ENV="prod" or "dev"
- run "npm install && npm run dev" to start the server
